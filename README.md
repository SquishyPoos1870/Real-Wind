# 🌬️ Real Wind

**Real Wind** adds atmospheric wind ambience to Factorio so the world feels more alive, open, and natural.

Instead of Nauvis feeling completely still, wind can now move through the environment with subtle ambience, gusts, and storm-aware behaviour when used alongside compatible weather mods.

Hear the air move. Feel the weather shift. Make your factory world feel less static without changing gameplay balance.

## ⚙️ What It Does

Real Wind gives Factorio extra environmental atmosphere through wind-focused effects and ambience.

The mod is designed to make the world feel more alive without covering the map in messy visuals or adding heavy simulation overhead. Wind can feel calm, active, or stormy depending on conditions and settings.

When used with compatible Real series mods, Real Wind can also help drive a connected atmosphere system:

* **Real Rain** can make wind pick up during rain and storms
* **Real Smoke** can react to wind with stronger smoke drift
* **Real Steam** can react to wind with sideways steam pull and stormy vapour movement

Real Wind still works perfectly fine on its own if those mods are not installed.

## 🛠️ Settings

Real Wind includes settings so you can tune the atmosphere to match your factory and performance needs.

| Setting              | What It Does                                                 |
| -------------------- | ------------------------------------------------------------ |
| **Enable Real Wind** | Master switch for Real Wind effects                          |
| **Wind Strength**    | Controls how noticeable the wind effect feels                |
| **Wind Ambience**    | Controls wind sound and atmosphere intensity                 |
| **Gust Strength**    | Controls how strong wind gust moments feel                   |
| **Storm Boost**      | Lets storms push the wind harder when Real Rain is installed |
| **Debug Wind Stats** | Optional testing output for checking wind behaviour          |

---

## 🌬️ Wind Strength Options

| Option       | Description                                                   |
| ------------ | ------------------------------------------------------------- |
| **Subtle**   | Light background wind, best for clean vanilla-style factories |
| **Balanced** | Recommended default; noticeable but not distracting           |
| **Heavy**    | Stronger wind feel for more atmosphere                        |
| **Stormy**   | More dramatic wind behaviour for intense weather setups       |

## ⭐ Recommended Settings

For a clean natural look:

| Setting           | Recommended Value |
| ----------------- | ----------------- |
| **Wind Strength** | Balanced          |
| **Wind Ambience** | On                |
| **Gust Strength** | Balanced          |
| **Storm Boost**   | On                |

| Setting           | Recommended Value |
| ----------------- | ----------------- |

| **Gust Strength** | Low               |
| **Storm Boost**   | On                |

For a stronger cinematic atmosphere:

| Setting           | Recommended Value |
| ----------------- | ----------------- |
| **Wind Strength** | Heavy or Stormy   |

| **Gust Strength** | High              |
| **Storm Boost**   | On                |

## 🧪 Behaviour Notes

* Real Wind is designed to be atmospheric, not intrusive.
* Wind should feel stronger during storms when Real Rain is installed.
* Smoke can drift harder during gusts when Real Smoke is installed.
* Steam can pull sideways during windy or stormy moments when Real Steam is installed.
* The mod is designed to stay lightweight and practical for normal gameplay.
* Real Wind is visual and ambience-focused and does not change recipes, pollution, damage, progression, or entity stats.

---

## 🔗 Compatibility

Real Wind is designed to work well by itself, but it can also optionally interact with other mods in the Real series.

### Works on its own

You do **not** need any other mod installed for Real Wind to work.

### Optional linked behaviour

If other compatible mods are installed, Real Wind can connect into the wider atmosphere system:

* **Real Rain**
  Rain and storms can increase wind strength, gusts, and atmosphere.

* **Real Smoke**
  Wind can help smoke drift more naturally, especially during storms.

* **Real Steam**
  Wind can pull steam wisps sideways and make vapour movement feel more natural.

This keeps the mod flexible:

**Great alone. Even better together.**

## 📌 Design Goal

The aim of Real Wind is to make Factorio’s world feel more alive without going over the top.

It is meant to sit somewhere between vanilla Factorio and a more immersive atmospheric world:

* More alive than vanilla
* More natural during storms
* Still clean enough for real factories
* Lightweight enough for normal gameplay
* Stronger when combined with the other Real mods
