Version: 1.0.6
Date: 2026-06-14
  Features:
    - Added optional Real Rain storm boost interface.
    - Rain/storms can now push Real Wind harder when both mods are installed.
    - Added remote wind state API for Real Smoke and Real Steam.
  Compatibility:
    - Still works fully stand-alone when Real Rain, Real Smoke, or Real Steam are not installed.

---------------------------------------------------------------------------------------------------
Version: 1.0.5
Date: 2026-06-15
  Changes:
    - Reworked forest bird ambience so it is emitted from actual nearby tree positions instead of the player.
    - Added distance-based bird volume so birds become louder when the player moves closer to trees and quieter when moving away.
    - Split the long bird ambience bed into shorter positional clips so the source position and volume can update more naturally while walking through forests.
    - Made gust whoosh sounds positional from nearby trees as well.
    - Kept all ambience Nauvis-only and tree-only.
  Notes:
    - This should make forest sound feel like it is coming from the trees, not from the camera or player body.

---------------------------------------------------------------------------------------------------
Version: 1.0.4
Date: 2026-06-15
  Changes:
    - Removed all dust gust visuals so Real Wind no longer creates ground dust/haze.
    - Removed the dust runtime setting and cleaned up the description text.
    - Made forest sounds much more reliable by checking ambience before the visual effect density roll.
    - Reprocessed the bird ambience to be louder and clearer in-game.
    - Increased the default forest ambience volume from 0.65 to 1.0 and allowed volume up to 2.0.
    - Made wind whoosh audio louder during gusts.
  Notes:
    - Forest ambience remains Nauvis-only and only plays near trees.

---------------------------------------------------------------------------------------------------
Version: 1.0.3
Date: 2026-06-15
  Features:
    - Made Real Wind effects Nauvis-only.
    - Expanded tree wind FX so every nearby tree prototype on Nauvis can receive wind overlays.
    - Added branch shimmer and stronger forest gust wisps.
    - Increased nearby tree scan radius and density for a fuller forest wind feel.
    - Added forest bird ambience using the supplied audio file.
    - Added soft wind whoosh ambience during gusts.
    - Added runtime settings for forest wind sounds and sound volume.
  Notes:
    - The mod still avoids global all-tree scans for UPS safety; it applies the effect to all tree types near connected players on Nauvis.

---------------------------------------------------------------------------------------------------
Version: 1.0.2
Date: 2026-06-15
  Features:
    - Added stronger tree wind FX to make forests feel more alive.
    - Added leaf sweep and canopy ripple smoke effects around trees.
    - Increased wind speed and gust strength across all intensity profiles.
    - Increased default effect density by lowering the default interval from 75 ticks to 60 ticks.
    - Added a runtime setting to enable or disable tree wind FX separately.
  Notes:
    - Factorio tree sprites cannot physically bend, so Real Wind now fakes movement with wind-driven overlay effects and leaf/debris motion.
